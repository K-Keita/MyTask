# My Task

1日のタスクを共有できるアプリです。

## URL

[https://mytask-7c4d2.web.app](https://mytask-7c4d2.web.app)

## 使用技術

- React.js(react-hooks)
- Redux
- firebase
- material-ui
- Sass

## 機能一覧
- アカウント登録、ログイン機能
- グループ機能
  - グループの新規作成
  - 既存グループへの参加
- タスクや起床時間を登録したカードの作成機能
- カードの共有機能
